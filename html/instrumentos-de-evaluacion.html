Instrumentos de evaluación

Para evaluar las actividades tanto la infografía como el podcast, se utilizarán instrumentos sencillos y variados que permitan conocer el progreso del alumnado. Para ello tenemos a continuación dos rúbricas para valorar las tareas, así como listas de cotejo y la observación directa en el aula. De este modo, se evaluará tanto el resultado como el proceso de aprendizaje.
